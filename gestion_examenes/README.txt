# Sistema de Calificación de Exámenes - Gestión de Exámenes

## Descripción

Este módulo corresponde a la gestión de exámenes dentro del sistema.
Se encarga de definir, organizar y configurar los exámenes mediante archivos de texto.

## Contenido de la carpeta

La carpeta `gestion_examenes/` contiene los siguientes archivos:

* `geExam.txt` → Archivo inicial con información general
* `estructuraExamenes.txt` → Estructura de los exámenes
* `configuracion.txt` → Configuración de los exámenes
* `asignacion.txt` → Asignación de exámenes a cursos
* `examenesProximos.txt` → Listado de exámenes próximos

## Proceso realizado

### 1. Inicialización del repositorio

Se creó el repositorio local con:
git init

### 2. Primer commit

Se agregó un archivo inicial:
git add File1.txt
git commit -m "primer commit - Se agrego File1"

### 3. Conexión con repositorio remoto

Se enlazó el repositorio local con GitHub:
git remote add origin <URL>

### 4. Solución de errores

* Error al hacer push por nombre de rama → se cambió a main
* Error por conflictos con repositorio remoto → se resolvió con:
  git pull --rebase origin main

### 5. Creación de rama

Se creó y utilizó la rama:
Karla-gestion_examenes

### 6. Desarrollo de la funcionalidad

Se creó la carpeta `gestion_examenes/` y sus archivos correspondientes.

### 7. Commits realizados

* Creación inicial de la carpeta y archivo principal
* Agregado de nuevos archivos con información de exámenes

### 8. Subida de cambios

Se subieron los cambios al repositorio remoto:
git push origin Karla-gestion_examenes

## Estado final

* Repositorio sincronizado con GitHub
* Rama de trabajo creada correctamente
* Archivos organizados y versionados
* Historial de commits completo

## Conclusión

Se aplicaron correctamente los fundamentos de Git, incluyendo el uso de ramas, commits, resolución de errores y trabajo con repositorios remotos.
